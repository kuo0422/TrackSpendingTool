# 記帳小工具

一個可以選擇日期並記錄是誰付錢的記帳小工具。

1. 連結Line以及google Apps Script，並將帳務明細紀錄在google雲端硬碟的excel中
2. 若今日尚未記帳，Line Bot會跳出通知訊息
